# 🎯 Campus Lost & Found Portal

## 📌 Project Name  
**Campus Lost & Found Portal** – A React‑based web app where students can post and browse lost/found items on campus.

## 📝 Description  
A React application that lets users:
- **Post** lost or found items (with photo preview).
- **Browse & filter** items by category, status, and date.
- **View item details** and contact the poster (mock chat).
- **Manage** their own posts (edit/delete).

All data is stored in **localStorage** (no backend required).

## 🎯 Goals
1. **React Fundamentals** – Components, hooks, routing, state management.
2. **UI/UX Design** – Clean, responsive interface using Tailwind CSS.
3. **Form Handling** – Image preview, validation, submission.
4. **Filtering & Search** – Dynamic filtering of items.
5. **Deployment** – Host on Vercel/Netlify for demo.
